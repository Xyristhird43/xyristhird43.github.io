🔭 Project Overview

This software identifies the shape of galaxies in images. It uses a computer vision model to look at images from the Galaxy10 DECals dataset. The program tells you if a galaxy is spiral, elliptical, or irregular. It automates the process of looking at large sets of space photos.

💻 System Requirements

To run this tool on your computer, ensure you meet the following conditions:

  • Windows 10 or Windows 11 operating system.
  • At least 8 GB of internal memory (RAM).
  • A modern processor (Intel i5 or AMD Ryzen 5 or better).
  • At least 2 GB of free disk space for the program and dataset files.
  • An internet connection for the first run to download necessary models.

📥 Download and Setup

Follow these steps to install and prepare the program on your computer:

  1. Visit the official release page to download the latest version.
  2. Choose the .zip file ending in "Windows-Installer".
  3. Once the download finishes, open your Downloads folder in Windows.
  4. Right-click the file and select "Extract All" to create a standard folder.
  5. Open the extracted folder and double-click the file named "setup.exe".
  6. Confirm by clicking "Run" if your computer asks for permission to open the file.
  7. Follow the on-screen instructions to finish the installation process.

🚀 How to Use

After you finish the setup, look for the Galaxy Classifier icon on your desktop. Open the application by double-clicking the icon. You will see a button to select a folder on your computer that contains galaxy photos. Click the "Start Analysis" button. The software will process each image and display the classification result next to the filename.

📐 Troubleshooting

If you encounter issues during installation, check that you are logged in as an administrator on your Windows computer. Ensure no other applications are using heavy resources when you run the analysis. If the program fails to find the target images, verify that your photos are in .jpg or .png format.
